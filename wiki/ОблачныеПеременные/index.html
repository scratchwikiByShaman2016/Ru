Облачные переменные

Облачные переменные — это переменные у которых значение у всех пользователей одинаковое. Их значение у всех изменяется. Если ни кого нет в проекте то значение сбрасывается до значения которое бвло во время сохранения проекта автором проекта.

Лимиты

До десяти облачных переменных в одном проекте

Значение до 250 символов.

Облачные переменные могут содержать только цифры и числа, до scratch 3.0 они могли содержать только положительные, когда появился scratch 3.0 (2019 год) они смогли содержать отрицательные числа

Интересные факты

С помощью ScratchApi можно получать и изменять их значения (а изменять их ещё можно и через X-токен даже через забаненый аккаунт). А ещё если получать значения облачных переменных и изменять их через Scratch Api то Скретч будет думать что проект открыт у одного пользователя!

Но текст можно шифровать в цифры чтобы текст можно было засунуть в облачную переменную
